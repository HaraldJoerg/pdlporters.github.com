PDL Mailing Lists

If you are interested in contact to other people using PDL, if you need help for a special problem or if you are a developer and want to help us improving PDL you should subscribe to one of our mailing lists. There are two lists (thanks to Frossie), one to talk about PDL and one for developers to coordinate their work.

  1. The general mailing list is perldl@jach.hawaii.edu. This list is intended for general user questions, non-technical discussions about PDL and announcements.

    To subscribe, send a message to perldl-request@jach.hawaii.edu containing a string in the following format:

    subscribe me@my.email.address

  2. The second list is the developer's mailing list. This can get very technical. The list address is pdl-porters@jach.hawaii.edu and the subscription address is pdl-porters-request@jach.hawaii.edu (send a message as above).

There are hypertext archives of both the perldl and pdl-porters mailing lists; thanks to Achim Bohnet.
