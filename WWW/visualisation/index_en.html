Visualisation Frontends

Scientific data often has to be visualized. PDL has interfaces to several external visualisation tools.

Packet Description
PDL::Graphics::TriD

OpenGL is the most popular library for 3D graphics today. The PDL installation scripts will look if you have installed the OpenGL libraries or the free free implementation Mesa and build the module if they are present.

PDL::Graphics::PGPLOT

PGPLOT is a very powerful fortran library for 2D plots with scientific quality. You need the PGPLOT library itself and the PGPERL perl module before compiling PDL.

PDL::Graphics::IIS Interface to the (somewhat old fashioned) SAOimage or Ximtool
PDL::Graphics::Karma Karma visualisation library. If you have installed it on your system the PDL module will be built.
