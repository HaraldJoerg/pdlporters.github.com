2004/12/28
PDL RESULTS IN THE NEWS: I am assembling a web page of science results, web pages, and interesting plots that were made with PDL. Some examples include the discovery of "ultrasound" on the Sun, and the presence of mature galaxies in the young Universe. If you have a nifty plot or a nice web page you'd like to show off, please send me a note: deforest (at) boulder (dot) swri (dot) edu. (CD)

2001/12/22
PDL 2.3.2 is available on CPAN. Merely a bugfix release to resolve a problem with gimp-perl. If you already got 2.3.1 just get the small patch. (CS)
2001/12/10
PDL 2.3.1 is available on CPAN. Lots of bug fixes and the new PDL::NiceSlice for easier slicing. Plus Inline::Pdlpp to write PDL::PP code on the fly... New windows binaries are also available for download. (CS)
2001/07/20
PDL::NiceSlice is available on CPAN. It introduces a much more convenient syntax for slicing piddles. (CS)
2001/07/14
A first attempt to compile PDL on Mac OS X has been largely successful. Check the report. (CS) Update: Binary packages of PDL for Mac OS X (+dependencies) are now avilable as part of the fink project.
