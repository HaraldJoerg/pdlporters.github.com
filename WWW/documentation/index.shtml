Documentation

PDL comes with online documentation. You can simply ask your perldl> command line for help and get information on every module and function included in the package, see PDL::Doc::Perldl. If you want to read the docs now you can use the online version.

The HTML documentation is extensive, but rather poorly organised at present. We are doing our best! Help is always welcome.

  • A quick overview is a great way to introduce yourself to PDL.
  • Before asking the team questions you should read the FAQ.
  • You can view the Online Documentation.
