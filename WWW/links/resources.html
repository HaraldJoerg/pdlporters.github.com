Useful resources

Here is a useful list of resources for PDL users.

Perl itself has numerous resources, the first points of call must be The Perl Institute, whom we thank for `hosting' this site, and The Perl Home Page which contains extensive documentation and tutorials about Perl as well as being the jump point for the Comprehensive Perl Archive which organises the distribution of the many hundreds of useful Perl modules out there.

PDL depends on numerous external libraries for it's features, here is a full list. Perhaps the two most common ones are the MesaGL library (which provides a free OpenGL implementation driving the 3D graphics) and the PGPLOT library used in the publication-quality 2D graphics library.

PDL has a nice logo thanks to Terry Greenlaw.

We will list some more here when we have time...
