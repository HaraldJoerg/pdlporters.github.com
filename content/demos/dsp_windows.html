dsp_windows - DSP windowing

This demo illustrates the PDL::DSP::Windows module, which provides many window functions for digital signal processing 
use PDL::DSP::Windows;
$w = pgswin(multi=>[1,2]); # PDL::Graphics::Simple window
The Hamming window 
$dspw = PDL::DSP::Windows->new(50, 'hamming');
$dspw->plot($w); $dspw->plot_freq($w);

Output

The "exact" Hamming window 
$dspw = PDL::DSP::Windows->new(50, 'hamming_ex');
$dspw->plot($w); $dspw->plot_freq($w);

Output

The Hann window 
$dspw = PDL::DSP::Windows->new(50, 'hann');
$dspw->plot($w); $dspw->plot_freq($w);

Output

The Welch window

Output
