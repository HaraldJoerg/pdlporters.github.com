PDL Mailing Lists

Our mailing lists are the best way to get in touch with other PDL users, or with the PDL developers. If you need help with a special problem, or if you wish to contribute to PDL, please join! We have two mailing lists now recently being hosted at SourceForge:

Users list (Subscribe)

The "pdl-general" list is the best place to ask general user questions and is the primary means of support for the Perl Data Language. It also features non-technical discussions about PDL, announcements, etc.

  • Archives
Developers list (Subscribe)

The "pdl-devel" list is where developers work on the internals of PDL. Discussions often get quite technical. Join this list if you'd like to contribute to PDL.

  • Archives
Git Commit List (Subscribe)

The "Git Commit" list sends an announcement every time a change is committed to PDL's main git repository. Join this list if you'd like to be informed of every contribution to PDL. There is also an option for a daily digest.
