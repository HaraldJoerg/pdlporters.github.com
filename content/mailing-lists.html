PDL Mailing Lists

Our mailing lists are the best way to get in touch with other PDL users, or with the PDL developers. If you need help with a special problem, or if you wish to contribute to PDL, please join! We have two mailing lits:

Users list (Subscribe)

The "PerlDL" list is the best place to ask general user questions. It also features non-technical discussions about PDL, announcements, etc.

  • Archives
  • Old Archives (pre-2005).
Developers list (Subscribe)

The "PDL Porters" list is where developers work on the internals of PDL. Discussions often get quite technical. Join this list if you'd like to contribute to PDL.

  • Archives
  • Old Archives (pre-2005).
Git Commit List (Subscribe)

The "Git Commit" list sends an announcement every time a change is committed to PDL's main git repository. Join this list if you'd like to be informed of every contribution to PDL. There is also an option for a daily digest.
